# 🏎️ Car Dodger Game

A fast-paced car dodging game built entirely using **HTML**, **CSS**, and **Vanilla JavaScript**. No frameworks, no libraries – just pure frontend magic! 🚗💨

---

## 🎯 Features

- 🚘 Player car movement using arrow keys
- 🚧 Incoming enemy cars with randomized positions
- 💥 Collision detection & Game Over screen
- 🎮 Real-time score tracker
- ⚡ Smooth animations using `requestAnimationFrame()`
- 📱 Fully responsive design for desktop and mobile
- 🔒 Works offline — no internet needed

---

## 🕹️ Controls

| Key         | Action         |
|-------------|----------------|
| ⬅️ Left      | Move car left  |
| ➡️ Right     | Move car right |
| ⬆️ Up        | Move car up    |
| ⬇️ Down      | Move car down  |

---

## 💻 How to Play

1. Download or clone this repository.
2. Open the `index.html` file in any modern browser.
3. Use your arrow keys to control the car and dodge enemy vehicles.
4. Try to survive as long as possible — score increases with time!
